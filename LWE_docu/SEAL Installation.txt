--------------------------------------------------------------------------------------------
-------------------------------Win11+VS2022安装SEAL-----------------------------------------
--------------------------------------------------------------------------------------------
安装CMake至 C:/Program Files/CMake 

cmake version 3.26.3

通过命令行 cmake /V查看版本

# 需要将C:/Program Files/CMake/bin添加到环境路径中 （好像也不需要）
--------------------------------------------------------------------------------------------
安装SEAL 在csdn上找github链接
SEAL版本为 4.0.0
--------------------------------------------------------------------------------------------
以管理员身份打开Developer Command Prompt for VS 2022终端，进入SEAL文件夹，依次输入三行命令
cmake -S . -B build -G "Visual Studio 17 2022" -A x64
cmake --build build --config Release
cmake --install build
可以看到SEAL文件夹中 
...\build\lib\Release\seal-4.0.lib
.lib文件，以及C:\Program Files\目录下SEAL本地安装库，这样我们就完成了seal库的编译和安装

除了上面三行命令，剩下的步骤、在vs中的配置按照下面这个blog进行，可以成功：
https://blog.csdn.net/qq_46000760/article/details/123652221

如果要跑官方给的例子的话，把SEAL\native\examples里面的例子扔到项目就行
